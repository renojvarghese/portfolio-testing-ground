Breakout

Bounce a ball on the paddle to break the bricks in the sky. Some bricks break more easily than others, so it may take some time to clear a wave. Some upgrades will drop the bricks to help you out as well

A nice project to play with creating textures in Photoshop, 3D models in Maya, and a classic game. I was particularly proud of how I structured my code to be able develop new levels. I wish I could have taken some more time to create different textures to make each level more visually diverse.
