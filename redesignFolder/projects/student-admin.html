Student Admin WireFrame

Redesigning an old system

In my software engineering class, I worked alongside classmates to recreate the current Student Admin system where students see their grades, pay university bills, and search for classes. Taking on the role of designer, I made this annotated wireframe to show how students would search for classes and what a search result might look like. The dropdown menus, checkboxes, and search box narrow down a query, and each tab below represents one class returned from a query. Each tab prioritizes information a student would need, with class name most prominent and icons on the right to represent if a student can take the class or not.
