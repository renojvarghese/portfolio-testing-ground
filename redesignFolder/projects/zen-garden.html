CSS Zen Garden

Showing the power of CSS

The CSS Zen Garden is a challenge to take a given HTML file and only change the CSS. There have been hundreds of submissions throughout the years. I was impressed my many of the designs and able to create my little own CSS Zen Garden. The style is a old-time poster that informs the user about the CSS Zen Garden. For this project I made my own art assets in Adobe PhotoShop and Illustrator
