Scalar Vector Graphics

Teaching SVGs in a cool way

SVGs were always a mystery that I never understood until I was able to do a project based on them. The website teaches the basics of Scalar Vector Graphics and how to style and embed them in a website. The site uses Vivus.js which draws out SVG images as if they are being drawn on the screen. The BEST part about this site is at the very end, which incorporates Snap.svg, another JavaScript library. I was able to create an random snowflake generator that draws out a unique little snowflake everytime a button is pressed.
